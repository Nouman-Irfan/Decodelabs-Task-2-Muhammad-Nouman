## 📌 Project Overview

**Expense Tracker** is a simple and practical Python console application developed as **Task 2** during my **Decode Labs Internship**.

This application allows users to add expenses, view expense history, and generate a useful expense summary including total expenses, average expense, highest expense, and highest expense category.

---

## 🚀 Features

* ➕ Add new expenses with category and amount
* 📂 Store expenses permanently in a text file
* 📜 View complete expense history
* 🕒 Save date and time automatically
* 📊 Generate expense summary
* 💵 Calculate total expenses
* 📈 Calculate average expense
* 🏆 Find highest expense and its category
* ⚠️ Handle invalid input safely
* 🧩 Clean and beginner-friendly code structure
